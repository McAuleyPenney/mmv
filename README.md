
### mmv

A bash reimplemenatation of [itchyny/mmv](https://github.com/itchyny/mmv)
<br>
<br>

#### use
-------
- `mmv <filename1> ...`
- `mmv *`
<br>
<br>

#### install
-----------
Clone and source this function somewhere in your path
<br>
<br>

#### requirements
----------------
- `bash 4.0` or greater
